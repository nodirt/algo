Algorithms and data structures

Simple, asymptotically efficient implementations in Python

Implementation principle

To minimize asymptotic complexity, while maximizing code simplicity. Thus, constant factor is sacrificed for sake of code readability.

Sorting

Comparison sorting:
  • Quicksort and 3-way quicksort for input with duplicates
  • Merge sort
  • Heapsort
  • Insertion sort
  • Selection sort
Linear-time sorting:
  • Counting sort
  • Radix sort
  • Bucket sort
Other:
  • Sort randomly
  • Find i-th smallest element
